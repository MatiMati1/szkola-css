Co znaczy skrót ID? identification(identyfikacja)
Jaki jest cel stosowania ID? odróżnienie takich samych znaczników od siebie i wystylowanie każdego z nich w inny sposób
Jakie jest znaczenie znaku # stawia się go w css przed nazwą id
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? trzeba wstawić je do znacznika który chcemy odróżnić oraz nazwe id do css z poprzedzającym znakiem #
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.
  1. jedno id może być przypisanie do 1 elementu
  2. chcąc ostylować id w css przed nazwą należy dać znak #
  3. jeżeli id się powtórzy strona będzie działać ale jest to niepoprawne
